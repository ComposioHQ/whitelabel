#### Tech Stack:
- NextJS
- Composio JavaScript SDK

#### Run Locally

Clone the repo
```bash
git clone https://github.com/ComposioHQ/whitelabel.git
```

Run the server
```bash
cd whitelabel && npm run dev
```
